**EcoGuard Finance – Personal Finance Management App**
EcoGuard Finance adalah aplikasi manajemen keuangan pribadi berbasis mobile yang dirancang untuk membantu pengguna melacak arus kas,
menganalisis pengeluaran, dan memantau kesehatan finansial secara real-time dengan tampilan yang bersih dan intuitif.

**Fitur Utama:**
1. Visualisasi Arus Kas: Menampilkan ringkasan total saldo, pendapatan, dan pengeluaran dalam bentuk kartu dashboard yang informatif.
2. Analitik Pengeluaran Interaktif: Menggunakan diagram lingkaran (doughnut chart) untuk memberikan gambaran proporsi pengeluaran berdasarkan kategori secara visual.
3. Statistik Cerdas: Fitur deteksi kategori "Terboros", rata-rata pengeluaran harian, dan perhitungan Saving Rate otomatis untuk membantu pengguna tetap disiplin dalam menabung.
4. Manajemen Transaksi: Pencatatan histori transaksi yang mendetail dengan fitur filter berdasarkan pemasukan atau pengeluaran serta fitur pencarian cepat.
5. Antarmuka Responsif: Desain UI yang modern dan profesional, mengutamakan pengalaman pengguna (UX) yang memudahkan navigasi data keuangan yang kompleks.

**Teknologi yang Digunakan:**
1. Framework: Flutter (Dart)
2. UI Components: Custom Widgets, Material Design 3, Charts/Graphic library (seperti fl_chart)
3. State Management: (Sebutkan yang kamu gunakan, contoh: Provider / Bloc / GetX)\
4. Architecture: Clean Architecture / MVC
